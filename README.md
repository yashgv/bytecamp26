## 🛠️ Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18+) or your preferred runtime
- [Git](https://git-scm.com/)

### Installation

```bash
# Clone the repository
git clone https://github.com/yashgv/bytecamp26.git
cd bytecamp26

# Install dependencies (when added)
npm install
```

### Running the Project

```bash
# Development mode
npm run dev

# Build for production
npm run build

# Run tests
npm test
```
